## 🎯 What Is an Applied AI Engineer?

Most people get this wrong. Let's set the record straight:

```
❌ NOT this:  "Build GPT-4 from scratch"   (costs $500M+, takes 3 years)
❌ NOT this:  "Do math research on AI"      (that's a PhD researcher)

✅ THIS:  Take GPT-4 / Llama / Mistral → Customize it for a company
          Build RAG systems on company data
          Fine-tune models for specific business tasks
          Deploy AI products that run in production
          Make companies 10x more productive with AI
```

**Real example of what you'll build:**
```
Client: Hospital in Hyderabad
Problem: Doctors waste 3 hours/day reading patient history

You build:
  → Fine-tuned Llama 4 on 50,000 medical records
  → RAG system on hospital policy + drug databases
  → Multi-agent system: reads history → drafts notes → flags risks
  → Deployed on hospital's private server (no data leaves)

Result: Doctors save 2 hours/day. Hospital pays ₹25L/year for this.
Your fee: ₹8L for building it. ₹3L/year for maintenance.
```

This is what Applied AI Engineers do. This course teaches exactly that.

**Q: How much time do I need?**

| Timeline | Approach | Outcome |

| **16 weeks** (moderate) | 1.5–2 hrs/day | Job-ready with all 4 projects |

**Q: What background do I need?**

```
Minimum:  Basic Python (can write functions and loops)
Ideal:    1 year of any programming experience
Not needed: Mathematics, statistics, ML theory, cloud experience
```

**Q: Which module should I start with?**

```
Complete beginner    → Start at Module 1
Know Python basics  → Start at Module 3 (Prompt Engineering)
Have ML experience  → Start at Module 4 (RAG Systems)
```
